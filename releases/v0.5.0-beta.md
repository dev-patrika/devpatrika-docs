# Release: Conversational Assistant, Citation Engine, & Intelligence Engines (v0.5.0-beta)

This release implements a stateful RAG Conversational Assistant with session memory persistence and structured citations extraction, and introduces four new intelligence engines: Trending Topics frequency analysis, Weekly AI Reports, semantic Related Recommendations, and Tech Evolution Timelines.

---

## 🛠️ Key Deliverables

### 1. Persistent Conversational Bot & memory (`ChatMessage`)
* Stores session dialogue states inside a persistent SQLite `chat_messages` table.
* Loads latest history on requests to maintain context memory.

### 2. Context Citation Engine
* Runs parallel semantic lookups on Chroma DB (Wiki & News collections).
* Prompts the LLM (model router selects Groq `openai/gpt-oss-120b`, `qwen/qwen3.6-27b` or Google `gemini-2.5-flash`) to anchor responses using numeric references.
* Returns explicit list of citations with target URLs in JSON payload.

### 3. Trending Topics Engine
* Calculates concept mention metrics in recent articles and updates trend directions (`"up"`, `"down"`, `"stable"`).

### 4. Weekly AI Reports Compiler
* Synthesizes weekly developer updates (news, github repos, research preprints, topics) into an editorial markdown report.

### 5. Related Articles Recommender
* Utilizes a new `"news_items"` collection in Chroma DB to compute semantic similarity recommendations.

### 6. Technology Timeline Generator
* Compiles chronological stage progressions (Origin ➔ Adoption ➔ Production ➔ Growth) using local DB references.

* **`app/models/`**: Added `weekly_report.py`, `trending_topic.py`, and `chat_history.py` models.
* **`app/services/`**:
  * `vectorstore/chroma_service.py`: Added news collection support and related articles similarity recommenders.
  * `trending/trending_engine.py`: Mentions counter and SQLite trends sync.
  * `reports/weekly_compiler.py`: Weekly report compiler.
  * `wiki_curator/timeline_generator.py`: Technology stages compiler.
  * `chat/chat_service.py`: Conversational memory, RAG contexts, and citation extractors.
* **`app/routers/`**:
  * `ai.py`: Implemented `/api/ai/models` and `/api/ai/chat` endpoints.
  * `reports.py`: Exposed `/api/reports/weekly` list, details, and manual trigger compile endpoints.
  * `news.py`: Added `/api/news/{id}/related` endpoint.
  * `wiki.py`: Added `/api/wiki/{term}/timeline` endpoint.
  * `__init__.py`: Registered reports router.
* **`app/services/ingestion/scheduler.py`**: Integrated indexing, trending, and weekly reports compiler.
