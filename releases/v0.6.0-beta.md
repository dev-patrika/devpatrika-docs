# Release Notes: v0.6.0-beta (LangGraph Multi-Agent Workflows)

We are excited to announce `v0.6.0-beta`, introducing a major architectural shift from linear sequential scripts to stateful, loop-capable **Multi-Agentic Workflows** using **LangGraph**. Additionally, this release introduces layout flexibility, streamlined settings, and direct user feedback channels.

---

## 🚀 Key Features

### 1. LangGraph Multi-Agent Workflows
All background summarization, research paper translations, and glossary management tasks are now handled by independent, stateful LangGraph agents:
*   **DailyBriefAgent (`app/agents/daily_brief.py`)**: Handles ingestion loaders and structured AI news/GitHub summaries.
*   **WikiCuratorAgent (`app/agents/wiki_curator.py`)**: Automatically crawls new terms, checks PGVector similarity to avoid duplicate definitions, and executes an LLM merge prompt for overlapping concepts.
*   **ResearchDigestAgent (`app/agents/research_digest.py`)**: Downloads arXiv preprints, parses PDF text, chunks them recursively, and translates academic jargon into plain developer digests.
*   **ExplainWhyAgent (`app/agents/explain_why.py`)**: Autonomous RAG chat query router inside the conversation workspace.

### 2. Dedicated "Research Papers" Tab
*   Surfaced translated research digests in a dedicated **Research Papers** tab on the News Feed page, allowing developers to filter academic preprints separately from regular news.

### 3. Compact List View Toggle
*   Enabled switching between the default Grid (Tiles) view and a compact **List View** for quick scanning. User selection is persisted in `localStorage`.

### 4. Direct Feedback API via Brevo
*   Form submissions are now handled directly by a new authenticated endpoint `POST /api/feedback` that sends emails directly from the backend via Brevo, eliminating the need to open local email clients. Redundant user profile inputs (Name, Email) have been removed from the frontend contact form.

### 5. Plain English Summaries
*   Refactored LLM system prompts across all agents and pipeline engines to strictly enforce plain, straightforward, and direct English, preventing verbose or poetic prose.
